# Production bottlenecks detections.

Implement a feature that can anwser this question: Can I see bottlenecks by workstation in real time? This system accumulates worklog entries and average production time of an order. Example, time when order is relesed to production till a checkbox is ticked that it is manufactured.

Time averages can be used by product groups to identify how long it takes to manufacture same product type or group during peak or low manufacturing workloads.
