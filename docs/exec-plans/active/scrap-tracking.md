# Scrap tracking

Scrap tracking can be accomplished by tracking invoice amounts of sold metal scrap but that may not be always available.

The system supports cutting visualization with the recommended amount to be cut per number of selcted items either from one or combined with other orders.

The actual reported cut amount is usually slightly bigger, to prevent errors in a later manufacturing phases. Example, if system says that to manufacture 3 items of window sill elements only 2850mm of a coil length are required the floor reports 3000mm. This difference should be captured and used to build smarted analytices over material consumption in time.
